# AI量化交易功能页面说明

本项目为abu量化交易框架添加了AI量化交易功能的前端页面，包括功能展示页面和工作台页面，方便用户使用和测试新添加的AI量化交易功能。

## 页面介绍

### 1. AI量化功能展示页面
**路径**：`/ai-quant-features`
**菜单**：AI量化功能

该页面详细展示了所有实现的AI量化交易功能，包括：

- **因子工程模块**：基本面与另类数据因子扩展、因子正交化处理
- **深度学习模块**：LSTM时间序列预测、Transformer序列建模、CNN价格形态识别
- **集成学习模块**：Stacking集成学习、Bagging集成学习
- **因子生命周期管理**：完整的因子生命周期管理系统
- **策略权重分配**：基于信息论的策略权重分配
- **市场状态识别与适应**：智能市场状态识别

每个功能模块都包含详细的描述、特点列表和使用示例代码，帮助用户快速了解和使用这些功能。

### 2. AI量化交易工作台页面
**路径**：`/ai-quant-workbench`
**菜单**：AI量化工作台

该页面提供了实际操作界面，用户可以通过该页面测试和使用各种AI量化交易功能：

#### 功能模块

##### 2.1 因子计算
- 支持选择股票代码、因子类型和具体因子
- 提供基本面因子、另类数据因子和技术因子的计算
- 展示因子计算结果表格

##### 2.2 模型训练
- 支持LSTM、Transformer和CNN等深度学习模型
- 可配置训练天数、预测天数和输入特征
- 展示模型训练结果和预测结果图表

##### 2.3 因子正交化
- 支持PCA、线性回归和Gram-Schmidt等正交化方法
- 可配置方差阈值和选择需要正交化的因子
- 展示正交化结果和因子载荷矩阵

##### 2.4 市场状态识别
- 支持HMM、波动率聚类和市场周期识别等方法
- 可配置状态数量和市场指数
- 展示识别出的市场状态和当前市场状态

##### 2.5 策略权重分配
- 支持最大多样性、熵权法、信息比率和组合权重等分配方法
- 可配置风险容忍度和选择策略
- 展示权重分配结果饼图和组合性能指标

## 技术实现

### 前端技术栈
- Vue 3 + TypeScript
- Vite 构建工具
- 响应式设计，支持移动端访问

### 页面特点
- 现代化UI设计，美观易用
- 实时数据处理和结果展示
- 支持多种参数配置和功能选择
- 友好的用户交互体验
- 动画效果和过渡效果增强用户体验

## 使用方法

1. **访问页面**：在侧边栏菜单中点击"AI量化功能"或"AI量化工作台"即可进入相应页面

2. **功能展示页面**：浏览各个功能模块的介绍和使用示例，了解功能特性

3. **工作台页面**：
   - 选择功能选项卡
   - 配置参数
   - 点击执行按钮
   - 查看结果展示
